🎲 Dice Roll Simulator with Animation

A fun and interactive **dice rolling simulator** with real‑time ASCII animation!  
Watch the dice spin before revealing the final result. Supports 1–6 dice, custom dice types (D6, D20, etc.), roll history, and statistics.  
Built in **7 programming languages** – perfect for learning, game development, or just having fun!

## ✨ Features
- **Animated rolling** – watch the dice tumble in the terminal before the final result.
- **Multiple dice** – roll 1–6 dice at once (or more with custom settings).
- **Custom dice types** – choose D4, D6, D8, D10, D12, D20, or D100.
- **Sum display** – shows the total of all dice rolled.
- **Roll history** – stores the last 10 rolls with timestamps.
- **Statistics** – tracks total rolls, average, min, and max values.
- **ASCII art** – beautifully rendered dice faces (1–6) with borders.
- **Cross‑platform** – works on any terminal with ANSI support.
- **Interactive CLI** – easy‑to‑use menu with numbered options.

## 🚀 How to Run
Each file is standalone – run it with the appropriate interpreter/compiler.

| Language | Command |
|----------|---------|
| Python   | `python dice.py` |
| Go       | `go run dice.go` |
| JavaScript | `node dice.js` |
| C#       | `dotnet run` (or `csc Dice.cs && Dice.exe`) |
| Java     | `javac Dice.java && java Dice` |
| Ruby     | `ruby dice.rb` |
| Swift    | `swift dice.swift` |

## 🔧 Dice Types Supported
- **D4** – 1–4
- **D6** – 1–6 (default)
- **D8** – 1–8
- **D10** – 1–10
- **D12** – 1–12
- **D20** – 1–20
- **D100** – 1–100
